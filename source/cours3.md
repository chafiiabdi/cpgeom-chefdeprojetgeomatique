# Base de données & PostGIS

Cette partie rassemble les notions liées à PostgreSQL, PostGIS et à l'administration des bases de données spatiales.

## Objectifs

- Comprendre le fonctionnement d'une base relationnelle.
- Créer et organiser des tables.
- Écrire des requêtes SQL.
- Manipuler des géométries avec PostGIS.
- Créer des index spatiaux.
- Administrer une base de données géographique.

## Notions clés

| Notion | Utilité |
|---|---|
| **Table** | Stocker les données sous forme de lignes et colonnes. |
| **Clé primaire** | Identifier chaque enregistrement de façon unique. |
| **Clé étrangère** | Relier plusieurs tables entre elles. |
| **Géométrie** | Représenter des points, lignes ou polygones. |
| **Index spatial** | Accélérer les requêtes géographiques. |

## Requêtes à maîtriser

- `SELECT` pour consulter des données.
- `WHERE` pour filtrer les résultats.
- `JOIN` pour croiser plusieurs tables.
- `GROUP BY` pour agréger des valeurs.
- Fonctions PostGIS pour analyser les géométries.
