# VRT

Un VRT est un fichier virtuel qui référence des données existantes sans forcément les dupliquer.

## Usages

- Construire une mosaïque raster.
- Regrouper plusieurs fichiers.
- Organiser des chemins vers des sources.
- Tester une structure avant production.

## Points à vérifier

- Les chemins vers les données sources.
- La cohérence des projections.
- La disponibilité des fichiers référencés.
