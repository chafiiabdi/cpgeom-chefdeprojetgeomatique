# PyQGIS

PyQGIS est l'API Python de QGIS. Elle permet de manipuler des couches, lancer des traitements et automatiser des workflows.

## Exemples d'usages

- Charger une couche dans QGIS.
- Lire les attributs d'une couche.
- Lancer un outil de traitement.
- Exporter un résultat.
- Créer un script réutilisable.

```{tip}
Documente chaque script avec son objectif, ses entrées, ses sorties et les paramètres importants.
```
