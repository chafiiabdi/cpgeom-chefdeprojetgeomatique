# Type de gestion de projet

Cette page décrit les principales approches de gestion de projet et leur intérêt dans un contexte géomatique.

## Approches possibles

| Type | Principe | Usage adapté |
|---|---|---|
| Prédictif | Le périmètre, le planning et les livrables sont définis tôt. | Projet réglementaire ou cahier des charges stable. |
| Agile | Le besoin est affiné par itérations successives. | Application, prototype, outil interne ou besoin évolutif. |
| Hybride | Le cadrage est structuré, mais les livrables évoluent par cycles. | Projet SIG avec données, traitements et restitution. |
