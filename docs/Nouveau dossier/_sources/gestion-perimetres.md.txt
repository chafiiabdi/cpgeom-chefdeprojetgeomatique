# Périmètres du projet

Le périmètre précise ce qui est inclus, ce qui est exclu et les limites du projet.

## Éléments à définir

| Élément | Exemple |
|---|---|
| Objectif | Produire une base de données géographique fiable. |
| Zone d'étude | Commune, région, bassin versant, emprise projet. |
| Données incluses | Couches sources, référentiels, images, bases métiers. |
| Données exclues | Données non disponibles, hors zone ou non validées. |
| Livrables | Carte, base PostGIS, rapport, procédure, tableau de bord. |

```{tip}
Un bon périmètre évite les malentendus : il doit dire clairement ce qui sera fait et ce qui ne sera pas fait.
```
