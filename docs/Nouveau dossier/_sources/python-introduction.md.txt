# Introduction de Python

Python permet d'automatiser des tâches, manipuler des fichiers, traiter des données et préparer des scripts reproductibles.

## Bases à maîtriser

| Notion | Exemple |
|---|---|
| Variable | Stocker une valeur. |
| Condition | Exécuter selon un cas. |
| Boucle | Répéter une action. |
| Fonction | Regrouper une logique réutilisable. |
| Module | Importer des outils existants. |
