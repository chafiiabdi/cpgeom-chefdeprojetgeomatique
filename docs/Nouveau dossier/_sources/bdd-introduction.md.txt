# Introduction de la base de données

Une base de données relationnelle organise les informations sous forme de tables reliées entre elles.

## Notions clés

| Notion | Définition courte |
|---|---|
| Table | Ensemble de lignes et colonnes. |
| Champ | Colonne décrivant une information. |
| Enregistrement | Ligne correspondant à un objet ou événement. |
| Clé primaire | Identifiant unique d'une ligne. |
| Clé étrangère | Lien vers une autre table. |

## Requêtes à maîtriser

- `SELECT` pour consulter.
- `WHERE` pour filtrer.
- `JOIN` pour croiser des tables.
- `GROUP BY` pour regrouper.
