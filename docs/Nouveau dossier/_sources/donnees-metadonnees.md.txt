# Métadonnées

Les métadonnées décrivent une donnée afin de la comprendre, l'évaluer et la réutiliser.

## Champs utiles

| Champ | Description |
|---|---|
| Titre | Nom clair de la donnée. |
| Résumé | Description courte. |
| Source | Origine de la donnée. |
| Date | Date de production ou mise à jour. |
| Projection | Système de coordonnées. |
| Licence | Conditions d'utilisation. |
| Qualité | Limites et niveau de fiabilité. |
