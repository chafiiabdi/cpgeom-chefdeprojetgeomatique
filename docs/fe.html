Formats ESRI

Cette partie présente les formats SIG utilisés pour stocker, échanger et exploiter des données géographiques.

Objectifs

  • Comprendre les principaux formats de données SIG.

  • Identifier les avantages et limites du shapefile.

  • Utiliser une geodatabase lorsque le projet le nécessite.

  • Préparer des données pour l’échange entre logiciels.

  • Choisir un format adapté au contexte de production.

Formats à connaître

Format

Usage principal

Shapefile

Format historique très utilisé pour les données vecteur.

Geodatabase

Stockage structuré de données SIG dans l’environnement ESRI.

GeoPackage

Format ouvert pratique pour échanger des couches SIG.

CSV

Échange de données tabulaires, parfois avec coordonnées.

GeoJSON

Données géographiques adaptées au web.

Points de vigilance

  • Encodage des caractères.

  • Projection et système de coordonnées.

  • Longueur des noms de champs.

  • Types de géométries.

  • Poids des fichiers.

  • Compatibilité avec les logiciels utilisés.

Travaux à documenter

  • Conversion de formats.

  • Import/export depuis QGIS.

  • Vérification de projection.

  • Contrôle des champs après conversion.
