Documentation développée pour organiser les ressources, méthodes et bonnes pratiques utilisées dans les projets SIG et géomatiques.

Objectifs du site

  • Centraliser les connaissances et outils géomatiques.
  • Améliorer l'accès aux données, aux procédures et aux référentiels.
  • Structurer les contenus de formation de manière simple.
  • Faciliter la modification du site avec un fichier HTML basique.

Gestion de projet

Cette partie regroupe les méthodes pour cadrer, organiser, suivre et gouverner un projet géomatique.

Type de gestion de projet

Comparer les approches prédictive, agile et hybride selon le contexte du projet.

Périmètres du projet

Définir les objectifs, les limites, les livrables, les contraintes et les exclusions.

Répartition des tâches

Organiser les rôles, les responsabilités, les échéances et le suivi de l'avancement.

Risques

Identifier les risques, estimer leur impact et prévoir des réponses adaptées.

Gouvernance

Définir les acteurs, les décisions, les validations et les circuits de communication.
