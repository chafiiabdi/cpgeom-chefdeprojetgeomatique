# ISO 19115

La norme ISO 19115 encadre la description des métadonnées géographiques.

## Informations à documenter

| Information | Exemple |
|---|---|
| Titre | Nom de la donnée. |
| Résumé | Description courte du contenu. |
| Emprise | Zone géographique couverte. |
| Qualité | Niveau de fiabilité et limites. |
| Responsable | Producteur ou gestionnaire de la donnée. |
| Contraintes | Droits, licences et restrictions. |

```{tip}
Une métadonnée utile doit permettre à une autre personne de comprendre, évaluer et réutiliser la donnée.
```
