# Documentation CPGEOM

![Bannière CPGEOM](images/banner-cpgeom.png)

Bienvenue dans ma documentation de formation **Chef de projet géomatique**.

Ce site regroupe mes **notes de cours**, mes **exercices**, mes **fiches de révision**, mes **ressources techniques** et mes **travaux pratiques** autour de la géomatique, des bases de données spatiales, de la télédétection, de Python et de la gestion de projet.

---

## Objectif du site

L’objectif de cette documentation est de centraliser progressivement toutes les connaissances acquises pendant la formation **CPGEOM - Chef de projet géomatique**.

Elle me permet de :

- organiser mes cours par thématique ;
- garder une trace claire de mes apprentissages ;
- retrouver rapidement les notions importantes ;
- documenter mes exercices et mes projets ;
- construire un support professionnel réutilisable.
