# Cycle de données

Le cycle de données décrit les étapes de vie d'une donnée : production, validation, diffusion, mise à jour et archivage.

## Étapes

| Étape | Action |
|---|---|
| Collecte | Identifier et récupérer les sources. |
| Préparation | Nettoyer, convertir, structurer. |
| Contrôle | Vérifier qualité et conformité. |
| Diffusion | Partager dans un format adapté. |
| Mise à jour | Gérer les nouvelles versions. |
| Archivage | Conserver les versions utiles. |
