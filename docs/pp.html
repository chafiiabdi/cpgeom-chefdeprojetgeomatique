Python & PyQGIS¶

Cette partie présente l'utilisation de Python pour automatiser les traitements géomatiques et gagner du temps dans les tâches répétitives.

Objectifs¶

  • Comprendre les bases de Python.

  • Manipuler des fichiers et dossiers.

  • Utiliser des bibliothèques géomatiques.

  • Automatiser des traitements QGIS.

  • Créer des scripts PyQGIS.

  • Documenter les scripts produits.

Notions clés¶

Notion

Utilité

Variable

Stocker une valeur réutilisable.

Boucle

Répéter une action sur plusieurs éléments.

Condition

Adapter le traitement selon un cas.

Fonction

Réutiliser un bloc de code.

PyQGIS

Piloter QGIS avec Python.

Exemples de tâches automatisables¶

  • Renommer ou organiser des fichiers.

  • Charger des couches dans QGIS.

  • Lancer un traitement sur plusieurs couches.

  • Exporter des résultats.

  • Produire des cartes ou rapports.

  • Contrôler des données de manière répétée.

Travaux à documenter¶

  • Objectif du script.

  • Données d'entrée.

  • Étapes du traitement.

  • Résultat attendu.

  • Limites et améliorations possibles.
